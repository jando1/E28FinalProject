The file structure is as follows: 
task1.py: Implementation of the cubic spline pure pursuit without the whole slalom course. Generates random seed for gate position.
task2.py: Slalom course navigation controller w/ cubic spline pure pursuit only
task3.py: Slalom course navigation controller w/ cubic spline pure pursuit and particle fitler
pf_demo_for_yhja.py: particle filter test environment made by Professor Zucker. Imported into task3.py.
project4.py: contains necessary functions for code adapted from Project 4's code
project4task1.py: version of task1.py from project 4, using line instead of cubic
